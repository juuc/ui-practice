How It Works

Drag on each canvas to interact with individual rendering steps

1
Fold Line & Signed Distance
The fold line is the perpendicular bisector between the corner and the drag tip. Every pixel calculates its signed distance to this line.
Positive = corner side (paper lifted away)  |  Negative + reflected in bounds = covered by flap  |  Negative + out of bounds = flat page.
Flat page (dist < 0, no flap)
Flap coverage (dist < 0, reflected in bounds)
Corner lifted (dist > 0, background)
// fold line = perpendicular bisector of corner ↔ tip mid = (corner + tip) / 2 fn = normalize(corner - tip) // positive = corner side (lifted), negative = flat side dist = dot(pixel - mid, fn) // reflection: where does the flap pixel come from? reflPos = pixel - 2 * dist * fn
drag to move the fold line
2
Reflection-Based Texture Mapping
The flap's texture comes from reflecting each pixel across the fold line. reflPos = pixel - 2 * dist * fn gives the source coordinate. This guarantees the content matches perfectly when the page flattens.
// reflect pixel across fold line to get source coord reflX = px - 2 * dist * fn.x reflY = py - 2 * dist * fn.y // if reflected point is within page bounds → flap covers this pixel // sample texture at (reflX, reflY) — always matches when flat
drag to see reflection mapping in action
3
Shadow & Depth
Two shadow effects create depth: a crease shadow right at the fold line (paper curves away from viewer), and a cylinder gradient across the flap simulating how light falls on a curved surface.
// crease shadow: darkest right at the fold line crease = absDist < 15 ? 0.6 + 0.4 * (absDist / 15) : 1.0 // cylinder gradient: simulates light on curved surface cylinder = 0.5 + 0.5 * sin(t * π/2 + 0.2) finalColor = textureColor * crease * cylinder
drag to see shadow response
4
Backface Rendering
The folded flap shows the back side of the paper. The reflected texture is tinted grey-blue to distinguish it from the front, simulating real paper with a visible front/back difference.
// the entire flap is the backface (folded over) // tint reflected texture to look like back of paper color.r = src.r * 0.45 + 85 color.g = src.g * 0.45 + 80 color.b = src.b * 0.45 + 95 // slight blue tint
drag to reveal the backface
5
All Combined — Final Page Curl
All four steps combined: region classification → reflection mapping → shadow → backface. This runs per-pixel on the GPU every frame. In a shader, Reanimated shared values feed u_currPos directly on the UI thread — no JS bridge overhead.
// Per-pixel decision tree (runs on GPU) if (dist > 0) → background (paper lifted away) else if (reflected in bounds) → backface texture + shadow else → flat page (+ shadow near fold)
